# Http Nedir ?

- Http client ve server arasındaki haberleşme sırasında güvenli bir şekilde veri alışverişi yapmak için kullanılan protokoldür.Get,post,put,patch,delete

# Axios Nedir?

- Axios aslında fetch metodunun güncel halidir.BaseUrl tanımlama,api isteğinin ne kadar süre cevap bekleyeceğini tanımlama,istekleri birleştirme gibi özelliklere sahiptir.Axios'un bir diğer artısı client,server arasındaki haberleşmede JSON <-> Js dönüştürmelerini kendisinin otomatik yapmasıdır.Bu sayede daha kolay veri iletimi sağlayacağız.

# Dimensions

- Mobile uygulamalarda ekranda bulunacak tüm elemanların her cihazda en optimum şekilde konumlanması gerekir.Bunu yapabilmek için React Native içerisinde bulunan dimensions kullanılır.Bu yapı ilgili ekranın genişlik ve yüksekliğini ayarlayabilmemizi sağlar.
